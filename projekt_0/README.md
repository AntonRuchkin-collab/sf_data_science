### Описание проекта
Программа об угадывании случайно заданного числа в данном интервале

:arrow_up:[к оглавлению](https://github.com/AntonRuchkin-collab/sf_data_science/blob/main/projekt_0/README.md#Оглавление)

### Какой кейс решаем?
Реализуем алгоритм бинарного поиска

### Краткая информация о данных
Используется список из 10 000 случайных чисел

### Этапы работы над проектом
1. Реализация алгоритма бинарного поиска (АБП) для отдельного элемента - функция serch_elemnts()
2. Создание массива из 10 000 случайных числе, поиск каждого с помощью АБП
3. Запись количества итераций АБП в список и далее - вычисление среднего количества 
   итераций для всего массива чисел, что и является основным ответом в задаче - функция average_number()

### Результат
Результат работы алгоритма сходится с заданным числом и вычисляет среднее значение итераций на всём массиве чисел.

### Выводы
1. АБП успешно работает, можно заметить, что среднее значение итераций алгоритма очень устойчиво,
   фактически не зависит от самого набора случайных чисел и зависит только от границ интервала в которых они были подобраны, поэтому в этой задаче можно даже не использовать .seed();
2. Среднее значение итераций на интервале [1; 101] составляет 4 попытки;
3. Среднее значение итераций алгоритма поиска начинает превышать 20 попыток на интервале [1;     10000001]
